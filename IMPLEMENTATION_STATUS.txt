🎯 FORM SECTIONS & FIELDS (30+ Total)
═══════════════════════════════════════════════════════════════════════════════

1. Location / Basic Info
   └─ Case ID, Province, District, Union Council, Landmark
   └─ GPS: Latitude, Longitude
   └─ Assessment Date, Member Contact, Location Verified

2. Incident Information
   └─ Event Date, Type (Flood, etc.), Information Source

3. Infrastructure Information
   └─ Category, Type (Road/Bridge/Building)
   └─ Road Type, Bridge Type, Building Type

4. Damage Assessment
   └─ Extent (Minor/Moderate/Major/Severe)
   └─ Status (Fully/Partially/Non-Functional)
   └─ Estimated Cost (PKR), Confidence Level (Low/Medium/High)
   └─ Ownership (Public/Private/Mixed)

5. Community Impact & Response
   └─ Population Affected, Critical Impact
   └─ Immediate Actions, Support Required

6. Verification
   └─ Verified By (Name & Designation), Verification Date

7. Remarks & Additional Info
   └─ Remarks/Notes textarea

🔌 API ENDPOINTS

   GET /api/health

✅ Get All Assessments
   GET /api/assessments

✅ Create Assessment
   POST /api/assessments
   Body: { caseId, province, district, damageExtent, ... }

✅ Update Assessment
   PUT /api/assessments/{id}
   Body: { field1: value1, field2: value2, ... }

✅ Delete Assessment
   DELETE /api/assessments/{id}
